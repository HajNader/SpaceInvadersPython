# Space Invaders - Python Version

این پروژه یک نسخه کلاسیک و ساده از بازی Space Invaders است که با Python و Pygame ساخته شده است.

---

## پیش‌نیازها

- Python 3.x  
- pip (مدیر پکیج پایتون)  

---

قبل از اجرای بازی، مطمئن شوید Python و pip روی سیستم نصب هستند.
